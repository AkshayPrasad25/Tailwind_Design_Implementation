### Installation instructions

#### Prerequisites:
[1] Node must be installed.</br>

#### [1] Clone the repository
``` bash
git clone https://github.com/AkshayPrasad25/Tailwind_Design_Implementation.git
```

#### [2] Install required packages.
``` bash
npm i
```
#### You're all set!
